    WXJOURNEY

    WXJourney allows you to create a custom weather forecast by date and location.

    Forecast data is provided by the ECMWF and is available 15 days in advance.

    How to use:

    Add record(s) Enter a location using the [location picker]. Select a single date or date range in the [date picker]. Press [+] to create the record(s).
    Refresh forecast Use [refresh] to update the forecasts. New forecast data is available at 00:00, 06:00, 12:00, and 18:00 UTC.
    Change Temperature Unit Tap [°F °C] to toggle the displayed unit.
    Edit a record Change the date of an existing record by pressing the [date picker]. Records automatically sort chronologically.
    Delete a record Press [X] in an existing record to delete the record.
